# Shop, Save, Invest 🛍️💰📈

Turn everyday shopping savings into investments — automatically.

## What it does
- **Shopping & Sale Tax Savings** — Scan receipts or enter savings manually
- **Item Return Savings** — Log refunds and invest them before you spend
- **Auto-Detect Returns** — Connect your bank via Plaid to detect return credits
- **5 Risk Profiles** — High, Medium-High, Medium, Low, and Fixed rate

## Tech Stack
- React + Vite
- Recharts (portfolio charts)
- Mindee OCR (receipt scanning)
- Plaid (bank monitoring)
- Alpaca (paper trading)
- Supabase (user accounts)

## Getting Started

```bash
npm install
npm run dev
```

## Deploy
